# 线路管理

## POST 获取线路列表

POST /api/appsheets/sheetapi/getFormList

> Body 请求参数

```json
{
  "applicationid": "",
  "formheaduuid": "formTableId",
  "needpage": 1,
  "page": 1,
  "pagesize": 10,
  "sortid": "排序字段名",
  "isasc": "是否升序",
  "filters": [
    {
      "key": "字段名1",
      "value": "筛选值"
    },
    {
      "key": "字段名2",
      "value": "筛选值"
    }
  ]
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authorization|header|string| 否 |张佳鹏的token|
|body|body|object| 否 |none|
|» applicationid|body|string| 是 |none|
|» formheaduuid|body|string| 是 |none|
|» needpage|body|integer| 是 |none|
|» page|body|integer| 是 |none|
|» pagesize|body|integer| 是 |none|
|» sortid|body|string| 否 |none|
|» isasc|body|string| 否 |none|
|» filters|body|[object]| 否 |none|
|»» key|body|string| 是 |none|
|»» value|body|string| 是 |none|
|» fields|body|string| 否 |none|

> 返回示例

> 200 Response

```json
{
  "status": "string",
  "result": {
    "data": [
      "string"
    ]
  }
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» status|string|true|none||none|
|» result|object|true|none||none|
|»» data|[string]|true|none||none|
