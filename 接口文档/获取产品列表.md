# 产品目录

## POST 获取产品列表

POST /api/appsheets/sheetapi/getFormList

> Body 请求参数

```json
{
  "applicationid": "",
  "formheaduuid": "formTableId",
  "needpage": 1,
  "page": 1,
  "pagesize": 10,
  "sortid": "排序字段名",
  "isasc": "是否升序",
  "filters": [
    {
      "key": "字段名1",
      "value": "筛选值"
    },
    {
      "key": "字段名2",
      "value": "筛选值"
    }
  ]
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authorization|header|string| 否 |张佳鹏的token|
|body|body|object| 否 |none|
|» applicationid|body|string| 是 |none|
|» formheaduuid|body|string| 是 |none|
|» needpage|body|boolean| 是 |none|
|» page|body|number| 是 |none|
|» pagesize|body|integer| 是 |none|
|» sortid|body|string| 否 |none|
|» isasc|body|boolean| 否 |none|
|» filters|body|[object]| 否 |none|
|»» key|body|string| 是 |none|
|»» value|body|string| 是 |none|
